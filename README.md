# azure-cloud-docs
Repository containing all work about cloud focused on Azure

### Docs workspace README

### What’s in here

- **Rows = individual docs**. Open any row to read the full content.
- **Doc name**: The title of the document.
- **Category**: Tags that describe the doc type
    - Guide → Guide for week work,
    - Lab → Lab work,
    - Research → Research work,
    - Dictionary → Kind of dictionary explaining details
- **Number**: A simple grouping by week or "All".

---

### Useful views

- **All Docs**: Table focused on Cloud items, sorted by Week and Category.
- **Published Docs**: List of docs, newest first.
- **Docs by Category**: Table grouped by Category to scan related docs.
- **Board**: Simple table you can sort or filter as needed.

Tip: Use the view tabs to switch perspectives quickly. You can also add filters like Category = "Guide" or Week = "Week 1".

---

### How to read a doc

1. Open a row to see the page content.
2. Skim the **Doc name** and **Category** to understand intent.
3. Check **Number** for week context.
